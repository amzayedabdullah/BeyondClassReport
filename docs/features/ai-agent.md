# AI Learning Assistant (BeyondClass Agent)

The **BeyondClass Agent** is a core component of the platform, designed to provide personalized assistance and intelligent content generation for students.

## Features

### 1. Personalized Tutoring
Students can interact with the AI Agent to ask questions about course material, clarify complex concepts, and receive step-by-step guidance on assignments.

### 2. Intelligent Note-Taking
The agent helps students organize their thoughts by automatically summarizing key points from lessons and generating structured notes that can be saved to their profile.

### 3. Resource Generation (Studio)
The "Studio" feature allows students to generate additional learning resources from provided input:
- **Audio Overviews**: Listen to a synthesized summary of the lesson.
- **Video Overviews**: Visual summaries with key highlights.
- **Quizzes**: Test knowledge with AI-generated questions.
- **Flashcards**: Quick review cards for active recall.
- **Data Tables**: Structured summaries of technical data.

## Technical Integration
The agent is built using the **Vercel AI SDK** with the **Google Gemini Pro** model, ensuring fast and accurate responses. It is fully integrated with the Supabase backend for persistent conversation history and note storage.
