# Features Overview

BeyondClass offers a comprehensive suite of features designed to enhance practical learning:

1.  **Module-Based Learning System**: Structured learning paths focused on specific skills (Git, Linux, MERN, etc.).
2.  **AI-Assisted Assignments & Auto-Grading**: Interactive coding tasks with instant feedback powered by AI.
3.  **Skill Certificates & Portfolio Builder**: Automatic certificate generation and portfolio creation to showcase achievements.
4.  **AI Learning Assistant (BeyondClass Agent)**: Personalized tutoring and intelligent resource generation (Studio).
